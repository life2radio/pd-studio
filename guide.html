API 키 받기,
5분이면 끝

Claude AI를 이 앱에서 쓰려면 본인 API 키가 필요해요.
아래 6단계만 따라하시면 됩니다.

0

먼저 — Anthropic 계정 가입

이미 계정 있으시면 1단계로 넘어가셔도 됩니다.

🔗 console.anthropic.com 새 탭에서 열기
  • Google 계정 또는 이메일로 가입 (무료)
  • 전화번호 인증 1번 (스팸 방지용, 한국 번호 OK)
  • 신용카드 등록 안 해도 됨
1

좌측 메뉴에서 'API 키' 누르기

로그인 후 좌측 사이드바에서 API 키 항목을 찾아 누르세요. 표시된 ① 위치예요.

1
2

'+ 키 생성' 검은 버튼 누르기

오른쪽 'API 키 만들기' 박스 안에 검은 버튼이 보여요. 누르면 키 생성 팝업이 떠요.

2
같은 화면, 오른쪽 버튼이에요 ↑
3

키 이름 입력

팝업이 뜨면 워크스페이스는 Default로 자동 선택돼있어요. 그대로 두세요.

'키 이름 지정' 칸에 본인이 알아볼 이름 입력. 예: pd-studio

3
4

'추가' 검은 버튼 누르기

이름 입력하면 '추가' 버튼이 검게 활성화돼요. 누르면 키가 만들어집니다.

4
5

⚠️ '키 복사' 무조건 누르기

이 화면 닫기 전에 무조건 키를 복사하세요.
한 번 닫으면 다시 못 봐요. 만약 놓쳤으면 처음부터 다시 만들어야 합니다.

5

💡 복사 후 잠깐 메모장이나 카톡 '나에게 보내기'에 임시로 붙여놓으셔도 좋아요.

✓

PD 스튜디오에 붙여넣기 → 끝!

복사한 키 (sk-ant-...로 시작)를 PD 스튜디오 시작 화면 → API 키 입력란에 붙여넣고 저장하면 끝입니다.

막혔거나 안 되시면

  • 잔액이 $0으로 표시되면, 좌측 메뉴 '결제'에서 카드 등록 (최소 $5만 충전해도 충분)
  • 키가 작동 안 하면, 키 앞뒤 공백 없는지 확인 후 다시 붙여넣기
  • 'sk-ant-'로 시작하지 않으면 잘못 복사한 거예요
  • 그래도 안 되면 PD 스튜디오 첫 화면 → '자주 묻는 질문' 메뉴에서 추가 도움말 확인
